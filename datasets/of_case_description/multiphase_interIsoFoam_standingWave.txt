This simulation employs the interIsoFoam solver to model a 2D standing wave interaction between two immiscible, incompressible Newtonian fluids (water and air) under laminar flow conditions. The domain features slip boundaries for velocity on all walls and zero-gradient conditions for the phase fraction, with front/back boundaries set as empty. Gravity acts downward with a magnitude of 9.81 m/s². The initial state has zero velocity and uniform air phase (alpha.water = 0) throughout the domain, with the modified pressure field (p_rgh) initialized to zero.

The VOF interface capturing uses the isoAdvection method (reconstructionScheme isoAlpha) with phase bounding and clipping enabled. Time discretization applies the second-order Crank-Nicolson scheme. Divergence schemes include vanLeer for phase fraction convection and limitedLinearV for momentum convection. Gradients are computed using linear interpolation, while laplacian terms use linear corrected schemes. Pressure corrections require flux specification.

Solver settings employ a PIMPLE algorithm with momentum prediction disabled, performing 3 pressure-velocity correctors without non-orthogonal corrections. Linear solvers utilize preconditioned conjugate gradient methods for pressure fields (DIC preconditioning) and biconjugate gradient for velocity (DILU preconditioning), with relative tolerances of 0.05 for pressure and tight absolute tolerances (1e-7 to 1e-9) for all fields. Surface tension coefficient is 0.07 N/m, with water density 1000 kg/m³, air density 1 kg/m³, and respective kinematic viscosities 1e-6 m²/s and 1.48e-5 m²/s.
