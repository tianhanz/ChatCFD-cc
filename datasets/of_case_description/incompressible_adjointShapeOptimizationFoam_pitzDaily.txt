The pitzDaily case utilizes the adjointShapeOptimizationFoam solver with a k-epsilon turbulence model for incompressible turbulent flow simulation with adjoint-based shape optimization. It employs a steady-state formulation and solves for both primal (U, p, k, epsilon) and adjoint (Ua, pa) fields on a 2D domain with empty front/back patches. The inlet velocity is fixed at (10 0 0) m/s, while walls use no-slip conditions. Turbulence parameters include initial k=0.375 m²/s² and epsilon=14.855 m²/s³ with wall functions applied. 

Discretization uses Gauss linear schemes for gradients and corrected linear schemes for Laplacian terms. Divergence terms apply bounded Gauss upwind for convective fluxes. The solver employs the SIMPLE algorithm with GAMG for pressure fields (relaxation 0.3) and smoothSolver for velocity/turbulence (relaxation 0.7). Turbulence properties set kinematic viscosity to 1e-5 m²/s using Newtonian modeling. Adjoint boundary conditions include specialized adjointOutletPressure and adjointOutletVelocity at the outlet. The simulation aims to compute sensitivity fields for aerodynamic shape optimization while resolving steady turbulent flow characteristics.
