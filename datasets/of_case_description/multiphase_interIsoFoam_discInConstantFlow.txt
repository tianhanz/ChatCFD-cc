This simulation models a disc embedded within a constant two-phase flow field using the interIsoFoam solver. The setup is 2-dimensional and laminar, with no turbulence model activated. Two Newtonian fluids are present: water (density 1000 kg/m³) and air (density 1 kg/m³), both configured with zero kinematic viscosity (nu) and zero surface tension (sigma). Gravity acts downward in the y-direction with a value of -9.81 m/s².

The initial flow field specifies a uniform velocity of (1, 0, 0.5) m/s throughout the domain. Initially, the entire domain contains air (volume fraction of water, alpha.water, set to 0), and the relative pressure (p_rgh) is zero uniformly. Boundary conditions impose zero gradients for alpha.water, velocity, and p_rgh at the rim boundaries, while the front and back planes are empty (signifying a 2D case).

For discretization, the time derivative uses Euler scheme. Gradient terms use Gauss linear, divergence uses Gauss limitedLinearV (with coefficient 1) for momentum transport and linear for stress terms, and Laplacians use Gauss linear corrected. Interface capturing employs the PLIC (plicRDF) reconstruction scheme for the volume fraction equation, using an isoFace tolerance of 1e-10, a surfCell tolerance of 1e-10, and allowing up to 5 bounding iterations. The velocity field is held constant (frozenFlow), meaning only the interface evolves. No momentum predictor step is used. Pressure reference is set at cell 0 with a value of 0.
