This simulation employs the pisoFoam solver to perform a Large Eddy Simulation (LES) of the turbulent flow in the pitzDailyMapped case configuration. The k-equation (kEqn) LES model is utilized for turbulence closure. The fluid is Newtonian with a constant kinematic viscosity (nu) of 1e-05 m²/s. Initial fields for velocity (U), pressure (p), turbulent kinetic energy (k), and filtered viscosity (nuTilda) are uniformly set to zero throughout the domain. At the inlet boundary, velocity is mapped from an external source with an average value of (10 0 0) m/s, and turbulent kinetic energy (k) is also mapped with a value of 2e-05 m²/s². The outlet uses an inletOutlet condition. No-slip walls are implemented for velocity, while pressure uses a zeroGradient condition on walls. Front and back planes are set to empty.

Discretization settings include a backward time scheme (ddtSchemes). Gradients are computed using Gauss linear. The divergence scheme for velocity employs Gauss LUST grad(U), while turbulence fields (phi,k, phi,nuTilda, phi,B) use a Gauss limitedLinear 1 scheme. Laplacian terms and interpolation use Gauss linear corrected and linear schemes, respectively. Surface normal gradients use a corrected scheme.

The solution algorithm uses the PISO method with 2 momentum correctors and no non-orthogonal correctors. Pressure is solved using a GAMG solver with a tolerance of 1e-07 and a relative tolerance of 0.1. Variables U, k, B, and nuTilda are solved using a smoothSolver (Gauss-Seidel smoother) with a tolerance of 1e-05. The LES configuration specifically uses the cubeRootVol filter for the subgrid scale and includes vanDriest wall damping.
