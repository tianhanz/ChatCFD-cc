Using pimpleFoam to simulate a 2D turbulent flow with periodic boundaries and an oscillating inlet employing Arbitrary Mesh Interface (AMI). The fluid is incompressible with Newtonian viscosity ν=1e-6 m²/s. The k-epsilon turbulence model is activated. Key features include: 1) Inlet boundary applies a fixed velocity (1,0,0) m/s while oscillating vertically with 0.5m amplitude at 3.14 rad/s via dynamic mesh motion; 2) Outlet uses zero pressure gradient with pressureInletOutletVelocity; 3) Walls apply wall functions for turbulence parameters.

Discretization uses Euler scheme for time derivatives, Gauss linear for gradients (with cell-limited for velocity), linearUpwind for divergence, and limited corrected schemes for Laplacian terms. Solvers configure GAMG for pressure (relTol 0.01, tolerance 1e-5) and smoothSolver for velocity/turbulence fields (relTol 0.1, tolerance 1e-6). PIMPLE algorithm runs with 1 outer correction, 2 pressure correctors, and no non-orthogonal correction. Initial fields set velocity to zero, pressure to zero, k to 3.75e-3 m²/s², and epsilon to 1.8e-3 m²/s³.
