This simulation employs the simpleFoam solver for steady-state incompressible airflow around a simplified car model using the k-epsilon RANS turbulence model. The fluid is Newtonian with a kinematic viscosity of 1e-05 m²/s. The domain setup includes an inlet with fixed uniform velocity (10 0 0) m/s, a pressure-driven outlet, no-slip wall boundaries for the car body and surrounding walls using wall functions, and symmetry planes. Key discretization schemes utilize bounded upwind for momentum and turbulence convection (div(phi,U), div(phi,k), div(phi,epsilon)), Gauss linear for gradients, and Gauss linear corrected for Laplacian terms. Solution settings employ the SIMPLE algorithm for pressure-velocity coupling with pressure relaxation (0.3) and field relaxation for velocity/turbulence equations (0.7). Residual controls are set for pressure (1e-2), velocity (1e-4), and turbulence fields (1e-4). An explicit porosity source models flow resistance in a designated porous zone using Darcy-Forchheimer coefficients with a Cartesian coordinate system. Pressure is solved using GAMG with Gauss-Seidel smoothing and face-area agglomeration, while other fields use smoothSolver with symmetric Gauss-Seidel.
