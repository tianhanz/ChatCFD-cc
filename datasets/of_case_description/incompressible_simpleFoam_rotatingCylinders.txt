The rotatingCylinders case employs the simpleFoam solver for steady-state incompressible laminar flow simulation involving rotating cylindrical geometry. The configuration uses a Multiple Reference Frame (MRF) model where all cells rotate uniformly around the z-axis at 100 rad/s, while the outerWall boundary remains stationary. The innerWall has a no-slip condition consistent with rotation. Initial conditions set pressure and velocity fields uniformly to zero relative to the rotating frame, with frontAndBack boundaries declared as empty for a 2D simulation.

For spatial discretization, gradients use Gauss linear scheme, convection of velocity (div(phi,U)) applies bounded Gauss linearUpwind, and Laplacian terms use Gauss linear corrected. The solution applies the SIMPLE algorithm for pressure-velocity coupling without non-orthogonal correctors. Pressure is solved using GAMG with Gauss-Seidel smoothing, tolerance 1e-10, and relaxation factor 0.3. Velocity uses a smoothSolver with Gauss-Seidel, tolerance 1e-10, and relaxation factor 0.7. The fluid is Newtonian with kinematic viscosity set to 1 m²/s.
