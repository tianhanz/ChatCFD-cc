Simulate the two-dimensional laminar flow of a non-Newtonian fluid around an offset cylinder using the nonNewtonianIcoFoam solver. The fluid follows the CrossPowerLaw viscosity model with coefficients: reference viscosity nu0=0.01, infinite-shear viscosity nuInf=10, consistency index m=0.4, and power-law index n=3. The computational domain features an inlet boundary (left) with fixed velocity (1, 0, 0) m/s and zero pressure gradient, an outlet boundary (right) with fixed pressure (0 Pa) and zero velocity gradient, no-slip walls (top and bottom), and a no-slip cylinder surface. Initial conditions: uniform zero velocity internally and zero pressure throughout.

Discretization employs Euler time integration, Gauss linear gradient schemes (least squares for pressure), linear divergence scheme for velocity convection, and corrected surface-normal gradients. Pressure is solved using the GAMG method with relative tolerance 0.1 and absolute tolerance 1e-6, while velocity uses the smoothSolver with symmetric Gauss-Seidel smoother and absolute tolerance 1e-5. The PISO algorithm executes 5 correction steps per iteration.
