This simulation employs the shallowWaterFoam solver to model the 2-dimensional flow of a shallow water body over a square bump. The setup accounts for Earth's rotation effects using a rotating frame with angular velocity set to Earth's rotation rate (7.292e-5 rad/s) and standard gravitational acceleration (9.81 m/s²) in the vertical direction.

The initial water depth (h) is uniformly 0.01 meters across the domain. The initial velocity field (hU) is set to a small uniform value (0.001 0 0) m²/s (representing discharge) primarily in the x-direction. The initial total height field (hTotal) is also set uniformly to 0.01 meters. The reference water level (h0) starts uniformly at 0 meters.

Boundary conditions define the flow domain: the inlet uses a fixed value for velocity with zero gradient for depth; the outlet uses a fixed value for depth and zero gradient for velocity; the sides use a slip condition for velocity and zero gradient for depth; frontAndBack boundaries are empty, consistent with 2D simulation.

The PIMPLE algorithm performs pressure-velocity coupling with 3 outer correctors and 1 inner corrector per time step. Momentum prediction is enabled, and no non-orthogonal correction is applied. The time discretization uses the Crank-Nicolson scheme (0.9 theta). Spatial divergence of momentum (phiv, hU) employs the LUST scheme with 'un' limiter for numerical stability. Other spatial terms (gradient, Laplacian) use linear Gauss schemes. Solvers for depth (h) and discharge (hU) fields employ preconditioned conjugate gradient (PCG/DIC) and iterative smoothing (smoothSolver/GaussSeidel), respectively, with specified tolerances (1e-10 absolute) and relaxation factors (0.01 to 0.1 relative tolerance).
