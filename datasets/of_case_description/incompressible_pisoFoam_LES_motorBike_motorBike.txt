This simulation employs the simpleFoam solver to model steady-state, incompressible airflow around a motorcycle geometry using the Spalart-Allmaras turbulence model. The fluid is Newtonian with kinematic viscosity ν = 1.5e⁻⁵ m²/s. Initial velocity is uniformly set to (20 0 0) m/s, pressure to 0 Pa, and turbulence variable nuTilda to 0.05 m²/s across the domain.

Boundary conditions include:  
- **Inlet**: Fixed velocity (20 0 0) m/s and fixed nuTilda (0.05 m²/s).  
- **Outlet**: inletOutlet for velocity and nuTilda with uniform (20 0 0) m/s and 0.05 m²/s fallback values.  
- **Motorcycle surfaces**: No-slip velocity and fixed nuTilda = 0.  
- **Lower wall**: Fixed velocity (20 0 0) m/s (simulating moving ground).  
- **Symmetry planes**: Applied to upper, front, and back boundaries.  

Numerical settings:  
- **Spatial discretization**:  
  - Gradient terms use limited linear Gauss schemes.  
  - Divergence schemes apply bounded localBlended linearUpwindV for momentum and upwind for turbulence.  
- **Linear solvers**:  
  - Pressure: GAMG with tolerance 1e⁻⁷ and 0.1 relaxation.  
  - Velocity/nuTilda: Gauss-Seidel smoother with tolerance 1e⁻⁸ and 0.1 relaxation.  
- **Under-relaxation**: Pressure (0.3), velocity (0.5), nuTilda (0.5).  
- **SIMPLE algorithm** corrects pressure-velocity coupling with 0 non-orthogonal correctors.  

The setup focuses on aerodynamic analysis with ground effect and symmetry simplifications.
