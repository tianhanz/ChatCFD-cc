Simulating laminar flow within a two-dimensional mixing vessel containing a rotating impeller using the pimpleFoam solver. The case employs Arbitrary Mesh Interface (AMI) for dynamic mesh handling between rotating and stationary zones. The fluid is an incompressible Newtonian fluid with kinematic viscosity set to 1e-05 m²/s. The rotating region ("rotor" cell zone) undergoes solid-body motion defined by rotation around the z-axis at an angular velocity of 6.2832 rad/s, while the "stator" zone remains stationary. Front and back boundaries are treated as empty due to the 2D simulation.

Initial fields for pressure and velocity are uniformly set to zero across the domain. Pressure boundaries on rotor/stator use zero gradient conditions, while AMI boundaries employ cyclicAMI patches for field continuity. Velocity boundaries on rotor/stator enforce no-slip conditions via movingWallVelocity. Turbulence properties are explicitly set to laminar, though placeholder fields (k, epsilon, nut) with wall functions remain in the case files.

Discretization uses Euler scheme for time derivatives. Gradient terms use Gauss linear discretization. Divergence of velocity flux employs the Gauss linearUpwind scheme with gradient-based limiter. Laplacian terms apply linear corrected discretization. Pressure-velocity coupling utilizes the PIMPLE algorithm with two outer correctors, one inner corrector, no non-orthogonal corrections, and pressure reference at cell zero with zero value. Solver settings include GAMG for pressure prediction and smoothSolver/GaussSeidel for velocity, both with relative tolerances of 0.01 during iterations and tighter final tolerances. No relaxation is applied to velocity equations.
