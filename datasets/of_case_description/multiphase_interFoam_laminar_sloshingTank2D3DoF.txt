The simulation uses the interFoam solver to model laminar sloshing in a 2D tank with 3 degrees of freedom motion. This multiphase case tracks water-air interactions using the VOF method. Water (density 998.2 kg/m³, kinematic viscosity 1e-06 m²/s) and air (density 1 kg/m³, kinematic viscosity 1.48e-05 m²/s) are Newtonian fluids with zero surface tension. Gravity acts vertically downward at 9.81 m/s². The tank undergoes prescribed solid-body motion via the SDA function featuring roll amplitudes between 0.1-0.2, heave amplitude 4, and sway amplitude 2.4.

Discretization employs Euler time stepping for ddt terms. Divergence schemes use vanLeer for both phase fraction (alpha) and momentum (rhoPhi,U) to capture sharp interfaces. Gradient terms use Gauss linear, and Laplacians use Gauss linear corrected. Pressure is solved using geometric-algebraic multigrid (GAMG) with diagonal incomplete Cholesky smoothing for p_rgh, while velocity uses Gauss-Seidel smoothing. The PIMPLE algorithm performs 2 pressure-velocity corrections per iteration without momentum prediction or non-orthogonality correction. Phase fraction solution includes 3 sub-cycles with interface compression factor 1.5. Initial conditions set water fraction, velocity, and modified pressure p_rgh uniformly to zero throughout the domain. Wall boundaries use fixedFluxPressure and movingWallVelocity conditions consistent with the tank motion.
