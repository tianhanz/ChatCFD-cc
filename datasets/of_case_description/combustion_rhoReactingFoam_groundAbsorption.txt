Conduct a laminar, compressible, multi-species reactive flow simulation using the rhoReactingFoam solver to study methane (CH4) absorption at a porous base surface in gravity. The domain features an inlet with fixed CH4 concentration (1.0), temperature (300 K), and vertical velocity (0.01 m/s); an outlet/sides with pressure boundaries; and a reactive base boundary implementing species sorption (Langmuir isotherm) and enthalpy sorption. Initial conditions: uniform pressure (101325 Pa), temperature (300 K), species mass fractions (O₂: 0.233, N₂: 0.767, CH₄: 0), and velocity (0 m/s). Gravity is set to (0, -9.8, 0) m/s². Chemistry and turbulence are inactive. Discretization uses Euler time schemes, Gauss linear gradients, LUST convection for momentum, bounded linear convection for species/energy, and GAMG/PBiCGStab solvers for pressure/velocity-species systems with PIMPLE algorithm (1 outer, 2 pressure correctors). Boundary-specific mass/energy sources are applied via fvOptions patches.
