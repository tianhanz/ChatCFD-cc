The simulation employs the buoyantBoussinesqSimpleFoam solver for steady-state incompressible buoyant flow around an igloo with embedded fridge structures, using the Boussinesq approximation to model density variations driven by temperature. The k-epsilon RANS model is activated for turbulence closure. The fluid properties correspond to a Newtonian fluid with laminar kinematic viscosity (nu) set to 1e-05 m²/s. Thermal parameters include a thermal expansion coefficient (beta) of 3e-03 K⁻¹, a reference temperature (T_ref) of 300 K, a laminar Prandtl number (Pr) of 0.7, and a turbulent Prandtl number (Pr_t) of 0.85. Gravity is applied in the negative z-direction with a magnitude of 9.81 m/s². Initial conditions define a stagnant flow field (U = 0 m/s everywhere), a uniform relative pressure field (p_rgh = 0 m²/s²), a uniform initial temperature field (T = 265 K), a uniform initial turbulent kinetic energy (k = 0.1 m²/s²), a uniform initial turbulent dissipation rate (epsilon = 0.01 m²/s³), and zero initial turbulent viscosity (nut = 0 m²/s). Boundary conditions enforce no-slip walls for velocity. Temperature boundaries set most walls to a fixed value of 265 K, while surfaces representing fridge components ('twoFridgeFreezers_seal_0' and 'twoFridgeFreezers_herring_1') are fixed at 303 K. Wall functions are specified for epsilon (epsilonWallFunction), k (kqRWallFunction), nut (nutkWallFunction), and thermal diffusivity (alphatJayatillekeWallFunction with Pr_t = 0.85). Pressure (p_rgh) on walls uses a fixedFluxPressure condition based on the buoyant density field (rho_k). Discretization schemes utilize a steady-state approach for temporal derivatives. Gradient terms employ Gauss linear schemes. Convection terms for U and T use the bounded Gauss upwind scheme, while the same scheme is applied to turbulence fields (k, epsilon). Laplacian schemes use Gauss linear limited corrected with a coefficient of 0.33, except for the displacement field. The SIMPLE algorithm is applied for pressure-velocity coupling with 2 non-orthogonal correctors and a reference pressure cell set to zero. Solver settings define tolerance and relaxation factors: pressure (p_rgh) uses PCG/DIC with tolerance 1e-8, relTol 0.01, relaxation 0.7; velocity (U), temperature (T), and turbulence scalars (k, epsilon) use PBiCGStab/DILU with tolerance 1e-7, relTol 0.1; velocity relaxation is 0.2, temperature relaxation 0.5, and turbulence scalars relaxation 0.7. Residual controls target p_rgh < 0.01, U < 1e-4, T < 0.001, and turbulence fields (k, epsilon) < 0.001.
