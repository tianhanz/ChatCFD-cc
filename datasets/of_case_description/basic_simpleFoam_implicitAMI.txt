This is a simulation of steady incompressible laminar flow using the simpleFoam solver. The fluid is Newtonian with a constant kinematic viscosity. The computational domain features an Arbitrary Mesh Interface (AMI) with the implicit coupling setting enabled on the AMI1 and AMI2 boundaries, allowing for non-conformal grid connections. A uniform inlet velocity of (1, 0, 0) m/s is applied at the left boundary, while pressure is fixed to zero at the right boundary. Top and bottom walls are treated as slip walls with zero velocity specified. The solution approach employs the SIMPLE algorithm with consistent pressure-velocity coupling. Discretization uses the steady-state formulation, Gauss linear schemes for gradients, bounded Gauss linear scheme for the divergence of velocity flux, and corrected linear schemes for Laplacian terms. Pressure is solved using the Preconditioned Conjugate Gradient (PCG) method with DIC preconditioner. Velocity is solved using a Gauss-Seidel smoothed solver. Relaxation factors of 0.9 for the velocity equation and 0.6 for the pressure field are applied to aid convergence.
