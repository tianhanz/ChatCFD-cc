The simulation employs the pisoFoam solver to model turbulent flow over a backward-facing step (PitzDaily geometry) using Large Eddy Simulation with the dynamicKEqn subgrid-scale model. This transient incompressible flow case involves a 2D domain where fluid enters through an inlet with prescribed turbulent velocity conditions and exits through an outlet with pressure-based boundary conditions. Walls use no-slip conditions for velocity.  

Initial conditions set velocity to zero, pressure to zero, and turbulence fields (k, s, nuTilda) to negligible values. The dynamicKEqn model uses a cube-root volume-based filter width with default coefficients. Newtonian fluid properties are defined with kinematic viscosity ν = 1e-05 m²/s.  

Discretization schemes:  
- Time: Backward implicit  
- Gradients: Gauss linear  
- Divergence: LUST for momentum (blended central/upwind), limitedLinear for turbulence scalars  
- Laplacian: Linear with non-orthogonal correction  
- Pressure solver: GAMG with Gauss-Seidel smoothing  
- Velocity/turbulence solvers: SmoothSolver with Gauss-Seidel  
PISO algorithm performs 2 pressure corrections per timestep without non-orthogonal correction.  

The setup aims to capture unsteady flow separation and vortex dynamics downstream of the step.
