Conduct a steady-state thermal diffusion simulation using the laplacianFoam solver to model heat conduction across coupled domains with implicit Arbitrary Mesh Interface (AMI) handling via non-blocking communication. The thermal diffusivity coefficient (DT) is set to 4e-05 m²/s. The computational domain features multiple solid regions with prescribed temperature boundaries: solid1_top and solid1_b_top maintain fixed temperatures of 1 unit, while solid2_bottom and solid2_b_bottom are fixed at 0 units. All other boundaries apply zero-gradient insulation conditions. A cyclicAMI interface couples adjacent regions, inheriting internal field values.

Discretization employs Gauss-linear schemes for gradient terms and Gauss-linear-corrected schemes for Laplacian operators, with snGrad correction for surface-normal gradients. The solver configuration uses a geometric-algebraic multigrid (GAMG) approach for the temperature field, applying processor agglomeration and Gauss-Seidel smoothing with an absolute tolerance of 1e-06. Two non-orthogonal correctors address mesh irregularities during solution iterations.
