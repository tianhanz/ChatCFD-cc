Simulate steady-state heat conduction in a 2D geometry consisting of two interconnected blocks using the laplacianFoam solver. The domain features two Cyclic Arbitrary Mesh Interface (AMI) boundaries to connect the blocks, alongside top and bottom walls with zero temperature gradient (insulated conditions), a fixed temperature value of 1 on the left wall, and a fixed temperature value of 0 on the right wall. Front and back boundaries are treated as empty to enforce two-dimensionality. The thermal diffusivity is constant at 4e-05 m²/s, and the initial temperature field is uniformly set to 1.

Spatial discretization employs the Gauss linear scheme for gradients. The laplacian term uses the Gauss linear corrected scheme for accuracy, particularly handling non-orthogonal meshes. The simulation uses the steady-state approach (ignoring time derivatives). The temperature field is solved with the Preconditioned Conjugate Gradient method, employing the Diagonal Incomplete Cholesky preconditioner, a tolerance of 1e-06, and no relative tolerance. Two non-orthogonal corrections are applied per iteration.
