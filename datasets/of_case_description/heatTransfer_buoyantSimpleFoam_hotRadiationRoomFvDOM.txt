Employ the buoyantSimpleFoam solver to simulate steady-state buoyancy-driven flow with radiative heat transfer in a room containing a heated box using the k-epsilon turbulence model and the Finite Volume Discrete Ordinate Method (fvDOM) radiation model. The domain includes boundaries for floor, ceiling, fixed walls, and a heated box, with gravitational acceleration set vertically downward at 9.81 m/s². Thermophysical properties specify air as an ideal gas with constant dynamic viscosity (1.8e-05 Pa·s), constant specific heat (1000 J/kg·K), and Prandtl number 0.7. Initial temperature is uniformly 300 K except for the box surface at 500 K, while initial velocity is stationary. Pressure uses a Boussinesq approximation (p_rgh) with reference pressure 100,000 Pa. Radiation settings include 3 azimuthal and 5 polar divisions, constant absorption coefficient 0.5 m⁻¹, and blackbody surface boundaries with emissivity 1.0. Discretization uses Gauss linear gradients, steady-state time derivatives, and bounded upwind convection for momentum and energy. Linear pressure-velocity coupling with SIMPLE algorithm applies under-relaxation (U:0.2, p_rgh:0.7, k/epsilon:0.5) and residual controls for pressure (1e-2), velocity/energy (1e-3). Turbulence wall functions enforce no-slip velocity boundaries and wall-modeled epsilon/k. Thermal diffusion uses a corrected linear scheme with wall functions for turbulent thermal diffusivity.
